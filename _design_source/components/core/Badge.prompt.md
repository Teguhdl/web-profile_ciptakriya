Small uppercase IBM Plex Mono pill for status, project categories and metadata labels.

```jsx
<Badge tone="brand">Mekanikal</Badge>
<Badge tone="success" dot>Selesai 2025</Badge>
<Badge tone="outline">SCG Readymix</Badge>
```

Tones: `neutral`, `brand` (blue), `green` (leaf), `success`, `warning`, `danger`, `solid` (filled blue), `outline`. Set `dot` for a leading status dot.
