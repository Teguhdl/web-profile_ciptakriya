Pill-shaped brand button — brand-blue primary CTA, with leaf-green, outline, ghost and inverse variants for use on dark/photo surfaces.

```jsx
<Button variant="primary" size="lg" iconRight={<span>→</span>}>Hubungi Kami</Button>
<Button variant="outline">Selengkapnya</Button>
<Button variant="inverse" href="#kontak">Get a Quote</Button>
```

Variants: `primary` (blue, brand glow), `secondary` (leaf green), `outline` (steel border → blue on hover), `ghost` (text-only), `inverse` (white, for navy/photo backgrounds). Sizes: `sm` / `md` / `lg`. Use `block` for full-width, `iconLeft`/`iconRight` for icons, `href` to render an anchor.
