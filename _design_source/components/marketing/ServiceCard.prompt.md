Capability card — tinted icon tile, hover lift, and a blue→green edge bar that wipes in on hover. Use for the "Bidang Usaha / Services" grid.

```jsx
<ServiceCard
  index="01"
  icon={<i data-lucide="cog"></i>}
  title="Konstruksi Mekanikal"
  description="Fabrication, erection and dismantling of heavy mechanical structures."
/>
```

Pass any icon node via `icon`. `index` adds a mono number in the corner.
